# Imagine X (iMX Protocol)

A limitless multimodal creative studio that transforms seeds of thought into rich stories, stunning visuals, and narrated experiences using Gemini 3.

## Features
- Visual Storytelling: Upload images to generate engaging narratives.
- Multi-speaker Narration: Atmospheric TTS for every story.
- Neural Vault: Locally stored gallery of your imaginations.
- Grounded Knowledge: Integrated search grounding for factual narratives.
